# TASK: Sidebar UI Polish (Artist / Designer / Gallery Modules)

## GOAL
Vizualno i UX uskladiti sidebar tako da jasno prikazuje module po grupama (Artist Tools, Designer Tools, Gallery Tools), te poboljšava upgrade putanju i opći izgled. Ovaj zadatak ne mijenja logiku entitlements, samo UI.

---

# 1) ADD GROUP TITLES (SECTION HEADERS)

U sidebar dodati tri grupna naslova:

### ARTIST TOOLS  
– prikazuje se samo ako user.artist_access = true  
– ako user.artist_access = false → ne prikazivati grupu

### DESIGNER TOOLS  
– prikazuje se uvijek  
– ako user.designer_access = false → modul je zaključan

### GALLERY TOOLS  
– prikazuje se uvijek  
– ako user.gallery_access = false → modul je zaključan

**Style za group title:**
```
font-size: 11px;
font-weight: 600;
letter-spacing: 0.8px;
text-transform: uppercase;
color: #7D8CB5; /* soft navy-gray */
margin: 18px 0 6px 18px;
```

---

# 2) IMPROVE LOCKED MODULE DESIGN

Za svaki modul bez entitlementa:

### ✖ Current:
– samo lock ikona i cijena

### ✔ Required:
Prikazati:

- ikona modula (siva varijanta)
- naziv modula (disabled state)
- lock ikona desno
- ispod toga tanki caption:

`Upgrade to unlock Designer Tools`

**Disabled state styling:**
```
opacity: 0.55;
cursor: pointer;
```

**Hover state (i dalje zaključano):**
```
background: rgba(40,53,147,0.06); /* soft navy tint */
border-radius: 8px;
```

Klik otvara Upgrade modal:
```
openUpgradeModal("designer" or "gallery")
```
