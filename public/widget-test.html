How the embed code works:

<script src="https://your-roomvibe-domain/widget.js" data-artist-id="1" data-artwork-id="123"></script>

The widget automatically:
  • Creates a styled "See in your room" button
  • Opens RoomVibe Studio in a modal when clicked
  • Pre-loads the specified artwork if data-artwork-id is provided
  • Works on any external website (Shopify, Squarespace, etc.)
